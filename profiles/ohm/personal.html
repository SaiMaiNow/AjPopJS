ประวัติการเขียนโค้ด

ผมเริ่มได้เขียนโค้ดครั้งเเรกตอน ม.1 เพราะว่าไปเข้าชมรม Robot ผมได้เขียนโค้ด Arduino ได้รู้จักตัวเเปรง ได้รู้จักการเขียน loop เขียน function ต่างๆ

เเละหลังจาก ม.3 ผมได้เริ่มเขียน server เกมเเละเปิดเซิฟเกมเป็นของตัวเองเเต่ไม่ได้ทำทั้งหมดผมมีเพื่อนด้วยทำด้วย เเต่สุดท้ายก็ปิดเพราะว่าทะเลาะกับเพื่อนเลยอยากจะเปิดเองเลยได้ทำเองทั้งหมดเเต่ก็ไม่ได้เปิดเเต่ได้พัฒนา logic มาพอสมควร

พอขึ้นมา ม.4 เปลี่ยน โรงเรียนมา ส่วนบุญโญปถัมภ์ ได้รับงานเปิดเซิฟเวอร์เกมมาเลื่อยๆ พร้อมกับเริ่มฝึกทำ เว็ปเเอปมาด้วย ได้ลองใช้ js nodejs react nextjs ต่างๆ

เเละได้ลงเเข่งเขียนโปรเเกรมระดับจังหวัดลำพูนกับโรงเรียนอื่นได้ลองใช้ภาษา C ในการเเข่งรอบเเรก รอบสองได้ใช้ C++ ในรอบที่สองจนเเข่งจบได้รองที่หนึ่งของจังหวัดมา

เเละผมก็รับงานรับเปิดเซิฟเกม เเละ รับทำเว็ปไปด้วย ปัจจุบันผมรับงานจาก บริษัทที่ทำเกี่ยวกับทำสังกัด tiktok สตรีมเมอร์ อยู่ครับ
